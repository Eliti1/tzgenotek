1. git clone https://github.com/Eliti1/tzgenotek  
2. cd tzgenotek  
3. ./docker.sh 
4. ./yc.sh
5. ./registry.sh  
3 и 4 этапы отдельно, потому что там после установки перезапускается shell, если всё в 1 скрипт делать, то скрипт падает.
